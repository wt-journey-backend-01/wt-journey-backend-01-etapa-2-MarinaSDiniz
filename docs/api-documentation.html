API do Departamento de Polícia

Documentação da API para gerenciamento de agentes e casos policiais

Versão: 1.0.0

Informações Gerais

URL Base: http://localhost:3000

Formato: JSON

Autenticação: Não requerida (desenvolvimento)

Content-Type: application/json

Endpoints - Agentes

GET /agentes Lista todos os agentes
Agentes

Descrição

Retorna uma lista com todos os agentes do departamento de polícia.

Códigos de Resposta

200 Lista de agentes retornada com sucesso 500 Erro interno do servidor

Exemplo de Resposta (200)

[ { "id": "401bccf5-cf9e-489d-8412-446cd169a0f1", "nome": "Rommel Carneiro", "dataDeIncorporacao": "1992/10/04", "cargo": "delegado" } ]
GET /agentes/{id} Busca um agente por ID
Agentes

Parâmetros

id (path, obrigatório): ID único do agente (UUID)

Códigos de Resposta

200 Agente encontrado 404 Agente não encontrado
POST /agentes Cria um novo agente
Agentes

Body da Requisição

{ "nome": "João Silva", "dataDeIncorporacao": "2024/01/15", "cargo": "agente" }

Campos Obrigatórios

  • nome: Nome completo do agente
  • dataDeIncorporacao: Data no formato YYYY/MM/DD
  • cargo: delegado | investigador | perito | agente | auxiliar

Códigos de Resposta

201 Agente criado com sucesso 400 Dados inválidos
PUT /agentes/{id} Atualiza um agente existente
Agentes

Parâmetros

id (path, obrigatório): ID único do agente (UUID)

Códigos de Resposta

200 Agente atualizado com sucesso 404 Agente não encontrado
DELETE /agentes/{id} Remove um agente
Agentes

Parâmetros

id (path, obrigatório): ID único do agente (UUID)

Códigos de Resposta

200 Agente removido com sucesso 404 Agente não encontrado

Endpoints - Casos

GET /casos Lista todos os casos
Casos

Exemplo de Resposta (200)

[ { "id": "f5fb2ad5-22a8-4cb4-90f2-8733517a0d46", "titulo": "homicidio", "descricao": "Disparos foram reportados às 22:33...", "status": "aberto", "agente_id": "401bccf5-cf9e-489d-8412-446cd169a0f1" } ]
GET /casos/{id} Busca um caso por ID
Casos

Parâmetros

id (path, obrigatório): ID único do caso (UUID)

POST /casos Cria um novo caso
Casos

Body da Requisição

{ "titulo": "Roubo de veículo", "descricao": "Veículo furtado na região central", "status": "aberto", "agente_id": "401bccf5-cf9e-489d-8412-446cd169a0f1" }

Campos Obrigatórios

  • titulo: Título do caso
  • descricao: Descrição detalhada
  • agente_id: ID do agente responsável (UUID)

status é opcional (padrão: "aberto")

PUT /casos/{id} Atualiza um caso existente
Casos

Parâmetros

id (path, obrigatório): ID único do caso (UUID)

DELETE /casos/{id} Remove um caso
Casos

Parâmetros

id (path, obrigatório): ID único do caso (UUID)

Esquemas de Dados

Agente

{ "id": "string (UUID)", "nome": "string", "dataDeIncorporacao": "string (YYYY/MM/DD)", "cargo": "delegado | investigador | perito | agente | auxiliar" }

Caso

{ "id": "string (UUID)", "titulo": "string", "descricao": "string", "status": "aberto | fechado | em_andamento", "agente_id": "string (UUID)" }

Erro

{ "status": "error", "statusCode": "number", "message": "string" }
